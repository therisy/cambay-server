# What is it?
A platform where you can share posts about Design and Software. You can get feedback on the posts you make

## Remaining Works
- like and dislike for comments
- images will be uploaded for posts

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
